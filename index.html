Para que la app funcione en GitHub Pages necesitas tu propia API Key de Anthropic. Es gratuita para empezar (USD $5 de crédito).

2. Crea una cuenta (o entra con Google)
3. Menú izquierdo → "API Keys"
4. Toca "Create Key"
5. Copia la clave y pégala arriba
